# Car Management System

혺Built a Spring based car rental management app using Maven, where admins can manage customers, cars and rentals.
혺Implemented Model-View-Controller pattern, used MySQL as the database, performed database operations using Spring JDBC.
혺Implemented Ajax to allow users to login, rent cars, return cars, etc.
혺Implemented RESTful web services to parse and response user requests.

Users can login as a customer or a staff, the customer can search, rent and return cars. The staff can manage all the customers and cars.
